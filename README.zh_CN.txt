Swift Train Webp Converter 插件
Swift Train Webp Converter 是一个 WordPress 插件，可将 JPEG 和 PNG 图像转换为 WebP 格式，以提高网站的加载速度。

功能
将 JPEG 和 PNG 图像转换为 WebP 格式。
可配置 JPEG 和 PNG 图像的质量级别。
可选择要转换的图像格式。
可以选择是否删除源图像。

安装
从 GitHub 下载 ZIP 文件。
在 WordPress 管理面板中选择“插件”>“添加新”。
选择“上传插件”，然后选择下载的 ZIP 文件。
点击“安装”。
激活插件。

使用
转换所有 JPEG 和 PNG 图像：在“设置”>“Swift Train Webp Converter”中单击“保存设置”。
选择要转换的图像格式：在“设置”>“Swift Train Webp Converter”中选择要转换的图像格式，然后单击“保存设置”。
配置质量级别：在“设置”>“Swift Train Webp Converter”中选择要使用的 JPEG 和 PNG 图像的质量级别，然后单击“保存设置”。
选择是否删除源图像：在“设置”>“Swift Train Webp Converter”中选择是否删除源图像，然后单击“保存设置”。

注意事项
本插件需要 PHP 支持 GD 库才能正常运行。
